# Button

대충 이러이러한 컴포넌트

## 지켜주세요

- 버튼의 세로 크기를 조정하지 마세요.
- 버튼 내 색상을 컬러 팔레트에 정의되지 않은 색상으로 정의하지 마세요.
- 한 페이지에서 동일한 버튼을 2개 이상 사용하는 행위를 지양해주세요.<br /><br />

## Variations

### Primary
사용 예시는 이하와 같습니다.
- 동작을 활성화하기 위한 버튼
- 링크 속성을 가지고 있는 경우

__Primary의 대표적인 사용처는 이하와 같습니다.__
- 링크 역할을 하는 페이지 내에서 사용
- 프로젝트 정보

__Primary mini의 대표적인 사용처는 이하와 같습니다.__
- 컴포넌트 내에서 사용하여야 하는 경우 (예시 : 후원 정보 컴포넌트) <br /><br /><br />
