# GREETING

안녕하세요! 본 문서는 __픽션네트워크(웹) 디자인 시스템__ 가이드라인 입니다.  
<br />

## 문서 구성  
### GUIDELINES  

- __COLORS__  
__픽션네트워크__에서 공용으로 사용되는 색상에 대하여 기재합니다.
- __FONTS__  
__픽션네트워크(웹)__에서 공용으로 사용되는 색상에 대하여 기재합니다.  

<br />
<hr />

### COMPONENTS
- __ATOMS__  
Button, Input 등의 요소를 __ATOMS__라고 하며, 최소 요소인 Element간의 조합은 __ATOMS__로 분류합니다.
- __MOLECULES__    
최소 1개 이상의 ATOMS가 포험된 조합 혹은 특수 케이스로, Grid의 영향을 받는 컴포넌트부터 __MOLECULES__로 정의합니다.  
<br /><br />
