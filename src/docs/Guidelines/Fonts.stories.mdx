# FONTS
픽션네트워크에서 사용되는 서체를 기재한 문서입니다.  
이하는 색상 가이드라인에 대한 규칙입니다. <br />

- 모든 서체의 전제는 __사용자가 문제 없이 인지 및 정독할 수 있도록__ 입니다.
- 가이드라인에 기재된 항목 이외의 서체 스타일이 2회 이상 사용되었을 경우 본 문서에 기재합니다.
- 서체에 적용된 모든 색상에 대한 배경과의 대비 비율을 최소 1.2:1을 유지해야 합니다. <br />
(이는 악조건의 디스플레이에서 최소한의 대비를 유지할 수 있는 대비 수치입니다.)
