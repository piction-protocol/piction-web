# COLORS
픽션네트워크에서 사용되는 색상을 기재한 문서입니다.  
이하는 색상 가이드라인에 대한 규칙입니다.

- 픽션네트워크의 모든 플랫폼에서 동일하게 사용되는 색상을 기재합니다. 
- 가이드라인에 기재된 항목 이외의 색상이 __2회 이상 사용__되었을 경우 본 문서에 기재합니다.
- 흰 색에 대비하여 모든 색상에 대한 대비 비율을 __최소 1.2:1__을 유지해야 합니다. (이는 악조건의 디스플레이에서 최소한의 대비를 유지할 수 있는 대비 수치입니다.)
- 파생 색상의 경우, 색상 이름의 뒤에 __--타입__을 기재합니다.
- 모든 색상은 소문자 표기를 원칙으로 합니다. (이는 후첨자도 동일합니다.)


## 첨자 사전
- strong : Original 색상 대비 강조된 색상을 의미합니다.
- pale : Original 색상 대비 흐린 색상을 의미합니다.
